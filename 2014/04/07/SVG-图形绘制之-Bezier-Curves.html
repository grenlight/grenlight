SVG 图形绘制之 Bezier Curves

07 Apr 2014

直线，折线，矩形这类直来直去的没啥可说的，主要是弧线，曲线的绘制

弧形（Arcs）

绘制弧形可以使用「A」或「a」命令：the uppercase command (A) uses absolute coordinates for its endpoint, where the lowercase command (a) uses relative coordinates (relative to the start point)

参数说明：The radius of the arc is set by the two first parameters set on the A command. The first parameter is rx (radius in x-direction) and the second is ry (radius in y-direction). Setting rx and ry to the same value will result in a circular arc. Setting rx and ry to different values will result in an elliptical arc. In the example above rx was set to 30 and ry to 50.

The third parameter set on the A command is the x-axis-rotation. This sets the rotation of the arc's x-axis compared to the normal x-axis. In the above example the x-axis-rotation is set to 0. Most of the time you will not need to change this parameter.

The fourth and fifth parameters are the large-arc-flag and sweep-flag parameters. The large-arc-flag determines whether to draw the smaller or bigger arc satisfying the start point, end point and rx and ry. Here is an example drawing 4 arcs each with different combinations of the large-arc-flag and sweep-flag:

//绘制两个相交的圆
//large-arc-flag 设置为0，表示按照可能的最小弧度绘制
d="M40,20  A30,30 0 0,0 70,70"
//large-arc-flag 设置为1，表示按照可能的最大弧度绘制
d="M40,20  A30,30 0 1,0 70,70"
//large-arc-flag 设置为1，sweep-flag 设置为1，表示按照可能的最大弧度镜像绘制
d="M40,20  A30,30 0 1,1 70,70"
//large-arc-flag 设置为0，sweep-flag 设置为1，表示按照可能的最小弧度镜像绘制
d="M40,20  A30,30 0 0,1 70,70"

 

贝塞尔曲线（Bezier Curves）

二次曲线(Quadratic Bezier Curves)命令：「Q」或「q」,

三次曲线(Cubic Bezier Curves)命令：「C」或「c」,

d="M150,150 C175,180 225,120 250,150"
d="M300,100 L350,100 a50,25 0 0,0 100,100 q50,-50 200,-170 Z"
